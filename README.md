## Features
- Multiple coding questions
- Hints and score system
- Real-time feedback
- Beginner-friendly

## Built With
- HTML, CSS, JavaScript (Frontend)

## How to Play
1. Choose a level
2. Solve the coding question
3. Submit your answer to see if it's correct!
